# Harleys in Hawaii — Interactive Lyric Sync Prank

A premium, mobile-first interactive lyric video experience with a delightful prank twist.

**Scroll to try syncing the lyrics yourself at 140 BPM** — or **let Naveen take over with mathematically perfect auto-sync**.

![Concept](https://via.placeholder.com/1200x630/1A1429/FF4D94?text=Harleys+in+Hawaii+Lyric+Prank)

## The Experience

- **Manual Challenge**: Scroll the long vertical lyrics and try to keep perfect sync with the song's groove. It's satisfying when you nail it... and intentionally tricky for humans.
- **Auto Mode by Naveen**: One tap and the page takes over — buttery smooth programmatic scrolling + precisely timed, bold, funky typography animations that hit every beat.
- Charming "struggle detection" that gently suggests switching to auto so you can actually enjoy the song.

Built as a fun, shareable web toy and portfolio piece. Perfect for mobile thumb-scrolling and viral clips.

## Tech Stack

- Vanilla HTML/CSS/JS (or Vite + Tailwind for production)
- Native Audio + requestAnimationFrame timing
- Lenis (optional) for ultra-smooth auto-scroll
- Designed for GitHub + Vercel static deploy

## Quick Start (Local)

```bash
git clone https://github.com/the-entertrainer/harleys-in-hawaii-lyric-prank.git
cd harleys-in-hawaii-lyric-prank
# For Vite version (recommended later)
npm install
npm run dev
```

Or just open `index.html` in a browser for the single-file starter.
